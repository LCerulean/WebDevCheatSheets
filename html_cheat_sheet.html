HTML Cheat Sheat

Elements

Structural Format Table

Most HTML elements follow the format of: <tag>element</tag>


Structual Elements

Tag Description
<html> Indicates that the document is HTML. It should come immediately after <!DOCTYPE html> and contain the rest of the contents of the file.
<head> Collection of metadata for the document (title, link, style, script). This is not visible to the user.
<body> Contains all content visible to the user.
<nav> Navigation, usually first thing inside body tag. Links or a list of links should be inside this tag.
<div> Divides content into sections. Not strictly necessary, but useful in some circumstances.
<h1>
<h2>...<h6>		 Headings, with h1 being the highest, and h6 the lowest.
<p> Paragraph of text.
<br> Line break. This does NOT use a closing tag.
<ol> Numbered list. List items must be inside list tags.
<ul> Bulleted list. List items must be inside list tags.
<li> List tag. Creates a numbered or bulleted list item (depending on if inside ordered or unordered list).
<img/> Insert an image. This does NOT use a second closing tag.
Use format: <img src="link/file" alt="text describing image"/>
<video> Inserts a video. Controls is necessary to include for user to start/stop the video.
Use format: <video src="link/file" controls>Text for "vidoe not supported"</vidoe>
<a> Link.
Use format: <a href="link address">Link Text</a>

Format Elements

Tag Description
<strong> or <b> Bold
<em> or <i> Italic
<small> Small
<mark> Marked/highlighted
<del> "Deleted" text/strikethrough
<ins> "Inserted" text/underlined
<sub> Subscript
<sup> Superscript

Table Elements

Tag Description
<table> Table. All table elements should be inside this.
<tr> Row. Contains the data for the row.
<th> Header cell. Should be inside first row.
<td> Data cell. Should be placed inside a row.
colspan Indicates how many columns a cell should occupy.
Use format: <td colspan="number">data<td>
rowspan Indicates how many rows a cell should occupy.
Use format: <td rowspan="number">data<td>
